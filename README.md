# vsdworkshop
Digital VLSI SoC Design and Planning Workshop (May 22,2024 to June 7,2024)

Tools used in the workshop

1.Openlane

2.Yosys and ABC

3.OpenRoad

4.Magic


Steps to be followed:

1. run_synthesis

2. run_floorplan

3. run_placement

4. run_cts

5. run_routing
